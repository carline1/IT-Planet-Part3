# IT-Planet-Part3

The final part of the International Olympiad IT Planet, OCR (Autumn 2021)

Model of visual search by pictures. 
Able to find similar images to the image transferred to the model.

The model is launched through main.ipynb.
To run the project, you need to download the unet_water_meter.hdf5 file from Yandex.Disk and put it in project root (it did not fit into the repository)<br>
Link to Yandex.Disk: https://disk.yandex.ru/d/TMIpNAxkJjyB5Q
